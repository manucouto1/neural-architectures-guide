# RWKV y RNNs Modernos - Frameworks

## Mamba

### mamba-ssm (Oficial)
- **Repo**: https://github.com/state-spaces/mamba
- **Autores**: Albert Gu, Tri Dao
- **Instalacion**: `pip install mamba-ssm`
- **Requisito**: CUDA (kernels custom obligatorios)

```python
from mamba_ssm import Mamba

# Capa Mamba individual
mamba_layer = Mamba(
    d_model=768,     # dimension del modelo
    d_state=16,      # dimension del estado SSM (N)
    d_conv=4,        # tamano del kernel de convolucion
    expand=2         # factor de expansion (d_inner = expand * d_model)
)

x = torch.randn(batch_size, seq_len, 768)
y = mamba_layer(x)  # (batch, seq_len, 768)
```

## Ejemplo: Entrenar un SSM desde Cero

```python
import torch
import torch.nn as nn
from mamba_ssm import Mamba

class SimpleMambaLM(nn.Module):
    def __init__(self, vocab_size, d_model, n_layers):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, d_model)
        self.layers = nn.ModuleList([
            nn.Sequential(
                nn.LayerNorm(d_model),
                Mamba(d_model=d_model, d_state=16, d_conv=4, expand=2),
            )
            for _ in range(n_layers)
        ])
        self.norm = nn.LayerNorm(d_model)
        self.head = nn.Linear(d_model, vocab_size, bias=False)
        # Weight tying
        self.head.weight = self.embedding.weight
    
    def forward(self, input_ids):
        x = self.embedding(input_ids)
        for layer in self.layers:
            x = x + layer(x)  # residual
        x = self.norm(x)
        return self.head(x)

# Modelo ~125M params
model = SimpleMambaLM(vocab_size=50257, d_model=768, n_layers=24)

# Entrenamiento standard
optimizer = torch.optim.AdamW(model.parameters(), lr=6e-4, weight_decay=0.1)
criterion = nn.CrossEntropyLoss()
```

## Tips Practicos

| Aspecto | Recomendacion |
|---|---|
| **GPU** | Mamba requiere CUDA. A100/H100 recomendado. RTX 3090/4090 funciona |
| **Precision** | BF16 para entrenamiento, FP16 para inferencia |
| **Batch size** | Grande (256+) para estabilidad. Gradient accumulation si no cabe |
| **Learning rate** | 3e-4 a 6e-4 con cosine schedule |
| **Warmup** | 1-2% del total de steps |
| **Mamba d_state** | 16 es default y funciona bien. 64 para tareas de memoria |
| **RWKV strategy** | `cuda fp16` para inferencia rapida |
| **Evaluacion** | Siempre comparar con Transformer del mismo tamano |
